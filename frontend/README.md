# Keuangan RT - Frontend (Vue 3 + Vite)

Aplikasi frontend pencatatan keuangan rukun tetangga berbasis Vue 3 dan Vite dengan JavaScript.

## Menjalankan Aplikasi Secara Lokal

### Install Dependensi
```bash
npm install
```

### Jalankan Mode Development
```bash
npm run dev
```

### Build untuk Produksi
```bash
npm run build
```

## Struktur Proyek
- Menggunakan `jsconfig.json` untuk konfigurasi modul JavaScript dan path resolution.
- `vite.config.js` untuk konfigurasi Vite dan PWA.
- `src/` untuk source code komponen dan halaman Vue.
